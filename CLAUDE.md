# StreamApp - Project Instructions

## Overview
Web-based OBS controller designed to function like an Elgato Stream Deck. Accessible from any device (iPhone, Surface, etc.) via browser.

## UI Guidelines
- Professional, clean design
- No emojis in the UI unless explicitly requested
- Mobile-first responsive design with large touch targets
- Dark theme preferred (typical for streaming software)

## Tech Stack
- ASP.NET Core 7.0 MVC
- C# backend
- obs-websocket-dotnet for OBS communication
- Bootstrap 5 / vanilla CSS for frontend
